# Swapper Toolkit

A Claude Code skill pack for DeFi operations: depositing funds, executing trades, and managing wallets.

## Skills

- **swapper-deposit** — Deposit and bridge assets into wallets and protocols

## Usage

Invoke skills via slash commands:

```
/swapper-deposit
```

## Development

Skills live in `skills/<skill-name>/skill.md`. Each skill is a self-contained markdown file with frontmatter metadata and instructions.
